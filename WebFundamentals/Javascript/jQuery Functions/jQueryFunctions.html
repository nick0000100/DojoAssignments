.click

Bind an event handler to the "click" JavaScript event, or trigger that event on an element.

  • .click(handler)
  • .click([eventData], handler)
  • .click()

.hide

Hide the matched elements.

  • .hide([duration][,complete])
  • .hide(options)
  • .hide()

.show

Display the matched elements.

  • .show([duration][,complete])
  • .show(options)
  • .show()

.toggle

There are two types:
Display or hide the matched elements.
Bind two or more handlers to the matched elements, to be executed on alternate clicks.

  • .toggle([duration][,complete])
  • .toggle(options)
  • .toggle(duration[,easing][,complete])
  • .toggle(display)
  • .toggle(handler,handler[,handler])

.slideDown

Display the matched elements with a sliding motion.

  • .slideDown([duration][,complete])
  • .slideDown(options)
  • .slideDown( [duration ] [, easing ] [, complete ] )

.slideUp

Hide the matched elements with a sliding motion.

  • .slideUp( [duration ] [, complete ] )
  • .slideUp( options )
  • .slideUp( [duration ] [, easing ] [, complete ] )

.slideToggle

Display or hide the matched elements with a sliding motion.

  • .slideToggle( [duration ] [, complete ] )
  • .slideToggle( options )
  • .slideToggle( [duration ] [, easing ] [, complete ] )

.fadeOut

Hide the matched elements by fading them to transparent.

  • .fadeOut( [duration ] [, complete ] )
  • .fadeOut( options )
  • .fadeOut( [duration ] [, easing ] [, complete ] )

.addClass

Adds the specified class(es) to each element in the set of matched elements.

  • .addClass( className )
  • .addClass( function )

.before

Insert content, specified by the parameter, before each element in the set of matched elements.

  • .before( content [, content ] )
  • .before( function )
  • .before( function-html )

.after

Insert content, specified by the parameter, after each element in the set of matched elements.

  • .after( content [, content ] )
  • .after( function )
  • .after( function-html )

.append

Insert content, specified by the parameter, to the end of each element in the set of matched elements.

  • .append( content [, content ] )
  • .append( function )

.html

Get the HTML contents of the first element in the set of matched elements.
Returns the content in the selected elements; first match
Set changes all not just first

  • .html()
  • .html( htmlString )
  • .html( function )

.attr

Get the value of an attribute for the first element in the set of matched elements.

  • .attr( attributeName )
  • .attr( attributeName, value )

.val

Get the current value of the first element in the set of matched elements.

  • .val()
  • .val(value)

.text

Get the combined text contents of each element in the set of matched elements, including their descendants.

  • .text()
  • .text( text )

.data

Store arbitrary data associated with the matched elements or return the value at the named data store for the first element in the set of matched elements.
Attach data to a div and retrieve it.

Look at these values { first: 16, last: "pizza!" }

  • .data( key, value ) Attach
  • .data( key ) Pull Data from the key
